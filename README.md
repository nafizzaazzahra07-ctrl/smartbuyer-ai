## 🌟 Overview
**SmartBuyer AI** is an AI-powered shopping assistant built to help users navigate the overwhelming world of online reviews. By aggregating authentic community sentiment from **Reddit, X (Twitter), Instagram, and TikTok**, our system generates a transparent **Trust Score** and balanced shopping reports.

According to recent e-commerce studies, **74% of online shoppers experience post-purchase anxiety** due to misleading information. For **Beauty & Skincare** enthusiasts, SmartBuyer AI provides personalized compatibility analysis based on your **Skin Type** and **Climate Conditions**—ensuring every purchase is a perfect match for your environment and needs.

---

## ✨ Key Features
* 🔍 **Multi-Source Sentiment Analysis:** Scrapes and synthesizes real-time opinions from global social platforms.
* 📊 **Smart Trust Score:** An AI-driven probability metric to detect biased, paid, or fake reviews.
* 🌿 **Personalized Skincare Logic:** Tailored ingredient safety analysis for Sensitive, Oily, Dry, or Combination skin based on local climate.
* 🌐 **Global Language Sync:** Seamlessly switch between **Indonesian and English** for both the UI and AI-generated content.
* 🛍️ **Trusted Marketplace Redirection:** Direct secure routing to verified official stores on **Shopee and Tokopedia**.
* 🎨 **Aesthetic Share Cards:** Export your analysis results into trendy, social-media-ready visual cards.

---

## 🛠️ Tech Stack & Architecture
* **Platform:** MeDo (Low-code Development Ecosystem)
* **Frontend Layer:** React, TypeScript, Vite, Tailwind CSS
* **Backend Engine:** Supabase (Authentication & PostgreSQL Database)
* **AI Integration:** OpenClaw API & Web Search (Real-time Sentiment Synthesis & Report Generation)
* **Localization:** i18next (Dual-language State Synchronization)
