## 🌟 Overview
**SmartBuyer AI** is an AI-powered shopping assistant built to help users navigate the overwhelming world of online reviews. By aggregating authentic community sentiment from **Reddit, X (Twitter), Instagram, and TikTok**, our system generates a transparent **Trust Score** and balanced shopping reports.

For **Beauty & Skincare** enthusiasts, SmartBuyer AI provides personalized compatibility analysis based on your **Skin Type** and **Climate Conditions**—ensuring every purchase is a perfect match for your environment and needs.

## ✨ Key Features
* 🔍 **Multi-Source Sentiment Analysis:** Scrapes and synthesizes real-time opinions from global social platforms.
* 📊 **Smart Trust Score:** An AI-driven probability metric to detect biased, paid, or fake reviews.
* 🌿 **Personalized Skincare Logic:** Tailored ingredient safety analysis for Sensitive, Oily, Dry, or Combination skin.
* 🌐 **Global Language Sync:** Seamlessly switch between **Indonesian and English** for both the UI and AI-generated content.
* 🛍️ **Trusted Marketplace Redirection:** Direct links to **Shopee and Tokopedia** (ID) or Amazon (US) for verified purchasing.
* 🎨 **Aesthetic Share Cards:** Export your analysis into trendy, social-media-ready image cards.

---

## 🛠️ Tech Stack
* **Platform:** MeDo (Low-code Development)
* **Frontend:** React, TypeScript, Vite, Tailwind CSS
* **Backend:** Supabase (Authentication & PostgreSQL Database)
* **AI & Search Engine:** MeDo AI Search & Web Search (Powered by **OpenClaw** for real-time sentiment analysis)
* **Intelligence Layer:** State-of-the-art LLM integrated via OpenClaw API for report synthesis
* **Localization:** i18next (Dual-language Synchronization)
